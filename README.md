# Python Data Science Toolkit Site

This GitHub Pages-ready folder contains:

- `index.html` — combined project generator and Jupyter shortcut guide
- `existing-project-setup.html` — generator for fresh and existing data science repositories
- `.nojekyll` — serves the site as plain static HTML

## Local test

Open `index.html` directly, or run:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

Push all files to the repository root. In GitHub:

1. Settings
2. Pages
3. Source: Deploy from a branch
4. Branch: main
5. Folder: / (root)
6. Save
